## Dependencies

The IP Royalty Pie Chart also requires [react-apexcharts](https://www.npmjs.com/package/react-apexcharts).

```bash
npm install react-apexcharts
```

```ts

      <IpRoyaltyPieChart />

  );
};

export default ExamplePage;
```
